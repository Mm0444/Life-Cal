# Quick Start Guide

## 🚀 Get Started in 3 Minutes!

### Step 1: Download & Install

```bash
# Clone the repository
git clone https://github.com/yourusername/lifecal-system.git
cd lifecal-system

# Build the project
mvn clean install
```

### Step 2: Run the Application

```bash
mvn javafx:run
```

### Step 3: Create Your Account

1. Click **"สร้างบัญชี"** (Create Account)
2. Fill in your information:
   - Username
   - Password
   - Weight & Height
   - Target Weight
   - Activity Level
3. Click **"ลงทะเบียน"** (Register)

### Step 4: Start Tracking!

✅ **Log your meals**  
✅ **Record exercises**  
✅ **Track weight**  
✅ **View progress**
