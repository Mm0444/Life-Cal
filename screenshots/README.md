## Screenshot Guidelines

- **Format**: PNG
- **Resolution**: 1920×1080 or higher
- **Quality**: High quality, clear text
- **Content**: Remove any personal/sensitive information
- **Naming**: Use lowercase with hyphens (e.g., `food-list.png`)

## How to Take Screenshots

1. Run the application
2. Navigate to the desired screen
3. Use screenshot tool (Windows: Win+Shift+S, Mac: Cmd+Shift+4)
4. Save to this directory with appropriate name
5. Update README.md if needed
